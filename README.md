# API для работы с технической документацией Спецдорпроект.
### Реализует следующий функционал:
1. Работа с паспортом светофорного объекта:
   - редактирование
   - сохранение в базу данных
   - получение ранее сохранённого паспорта
   - данные об изменениях по паспорту

## Локальная разработка

#### №Запуск приложения в docker:
По умолчанию приложение запускается на порту 8001
```commandline
    docker compose up
```

#### Запуск приложения uv:
```commandline
    uv run main.py
```

#### Запуск тестов скриптом:
```commandline
    ./start-compose-tests.sh
```
